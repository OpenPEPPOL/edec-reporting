== Reporting Period

The element `ReportPeriod` MUST contain the year and month of the reporting period.
This element SHALL NOT contain time zone information.

== Reporter Identification
The element `ReporterID` MUST contain rhe unique identification of the 
  Service Provider that is providing the data.
If the `schemeID` attribute is set to `CertSubjectCN` than the value must be
  the Peppol AP Certificate Subject CN (Common Name). 
  See <<codelist-repidscheme>> for the full code list.
